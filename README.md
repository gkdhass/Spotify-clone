# Spotify Clone

A Spotify-inspired music streaming web application built to replicate core features such as browsing music, playing tracks, managing playlists, and user authentication.

---

## Project Overview

This project is a full-stack Spotify clone that allows users to explore songs, albums, and playlists with real-time music data and smooth playback controls.  
It focuses on clean UI, responsive design, and practical API integration.

---

## Features

- User authentication using Spotify OAuth  
- Browse songs, albums, and playlists  
- Search functionality for music discovery  
- Music playback controls (play, pause, skip)  
- Create and manage playlists  
- Like and save favorite tracks  
- Fully responsive design for desktop and mobile  

---

## Tech Stack

- React.js  
- JavaScript  
- Spotify Web API  
- HTML5 & CSS3  
- Vite / Create React App
